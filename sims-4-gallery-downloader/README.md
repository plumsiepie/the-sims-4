TS4 Gallery Downloader (an anadius fork)

Original description:
"The Sims 4 - gallery downloader
UserScript that lets you download items from the gallery website.

To use this UserScript you need to install some browser extension first.
If you have Firefox, install Greasemonkey.
If you have Chrome, Opera, or anything else Chromium-based, install Tampermonkey or Violentmonkey.

To browse the gallery website you need to create a free EA account.

You can't search for items that use CC! And you can't upload anything either.

If you want a better solution - get a legit base game (it's free) and use one of my DLC Unlockers."
