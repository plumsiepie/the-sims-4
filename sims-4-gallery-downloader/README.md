	TS4 Gallery Downloader
A UserScript that lets you download households, lots, and rooms directly from The Sims 4 Gallery website.

Originally created by anadius.
This is an adopted fork maintained by plumsiepie after the original was retired.


Requirements
To use this script you need a browser extension that supports UserScripts:

Firefox — Greasemonkey
Chrome, Edge, Opera, or other Chromium-based browsers — Tampermonkey or Violentmonkey

You also need a free EA account to browse the gallery.

Limitations

You cannot search for or download items that use custom content (CC)
You cannot upload anything via the gallery website


Installation

Install one of the browser extensions listed above
Click the link below — your extension will prompt you to install the script automatically:

Install TS4 Gallery Downloader

Click Confirm install
Go to the gallery — a download button will appear on each item


Downloaded files
Files are saved as a .zip to your downloads folder. Extract the contents and place them in your Sims 4 Tray folder:
Documents\Electronic Arts\The Sims 4\Tray
